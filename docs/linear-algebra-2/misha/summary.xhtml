1.1 מטריצות דומות

יהיו A,B ∈ M n ( F ) , נאמר כי A ו- B דומות אם קיימת מטריצה הפיכה P כך ש- A= P -1 ⋅ B ⋅ P .
משפט: נתון A,B ∈ M n ( F ) ריבועיות, הבאים שקולים:
  1. A,B דומות.
  2. קיימת T:V → V ובסיסים C, C ′ של V כך ש- [ T ] C =A, [ T ] C ′ =B .
  3. לכל T:V → V , אם קיים בסיס C של V כך ש- [ T ] C =A , אז קיים בסיס C ′ של V כך ש- [ T ] C ′ =B .
ואם A,B דומות אז:
  1. Rank ( A ) = Rank ( B ) , N ( A ) = N ( B ) .
  2. tr ( A ) = tr ( B ) כאשר tr ( A ) = ∑ i=1 n ( A ) i,i .
  3. det ( A ) = det ( B ) .

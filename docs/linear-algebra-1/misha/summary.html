יהי V מ”ו מעל F , B בסיס של V . נסמן dim V=n , ויהי B=( b 1 , … , b n ) בסיס. על פי משפט, לכל v ¯ ∈ V קיימים ויחידים α 1 , … , α n ∈ F כך ש- v= ∑ i=1 n α i b i . נגדיר את הקואורדינטות של v ¯ לפי B להיות: [ v ¯ ] B =( α 1 ⋮ α n ) ∈ F n זה איזומורפיזם מ- V ל- F n . העתקת הקואורדינטות תסומן גם בתור [ ⋅ ] B :V → F n .

13.5 מרחב ההעתקות

הגדרה: Hom ( V,U ) ={ T ∈ U V ∣ T is linear } מרחב ההעתקות. זה תת מרחב של ⟨ U V ,+, ⋅ ⟩ .
משפט: dim ( Hom ( V,U ) ) = dim ( V ) ⋅ dim ( U ) - זה נכון אפילו אם V,U לא נוצרים סופית.

13.6 מטריציונית

הגדרה: לכל מטריצה A ∈ M m × n ( F ) , נגדיר את ההעתקה המטריציונית המתאימה ל- A , T A : F n → F m : T A ( v ¯ ) =A x ¯ פונקציה f נקראת מטריציונית אם קיימת מטריצה A ∈ M m × n ( F ) כך ש- f= T A , ונסמן A=[ f ] .
השיטה למצוא את המטריצה [ T ] היא: [ T ] =( T( 1 0 ⋮ 0 ) T( 0 1 ⋮ 0 ) … T( 0 ⋮ 0 1 ) )
משפט: תהא T: F n → F m , אזי T העתקה לינארית ⟺ T מטריציונית.
טענות:
  1. Sols ( A ) = T A -1 [ { 0 ¯ } ] = ker ( T A ) .
  2. C( A ) =Im( T A ) .
  3. T A על ⟺ עמודות A פורשות. אם היא ריבועית אז גם הפיכה.
  4. T A חח”ע ⟺ עמודות A בת”ל. כי אין שתי דרכים להגיע לאותו הדבר.
  5. T A הפיכה ⟺ עמודות A בסיס ⟺ A הפיכה.
  6. אם T,S: F n → F m מטריציונית אז T+S: F n → F m מטריציונית. יתר על כן [ T+S ] =[ T ] +[ S ] .
  7. אם T ∈ F n → F m מטריציונית ו- α ∈ F אז α ⋅ T: F n → F m מטריציונית. יתר על כן [ α ⋅ T ] = α ⋅ [ T ] .
  8. אם S: F m → F k ,T: F n → F m מטריציוניות אז S ○ T: F n → F k מטריציונית, ו- [ S ○ T ] =[ S ] ⋅ [ T ] )כפל המטריצות הוא ההרכבה(.

13.7 מטריצה מייצגת

13.7.1 הגדרות

הגדרה: תהא T:V → U צ”ל V,U נוצר סופית. יהי B בסיס של V , ו- C בסיס של U . נגדיר את ההעתקה המייצגת T C B : F dim ( V ) → F dim ( U ) : T C B = Q C ○ T ○ Q B [ T ] C B =[ T C B ] בעצם מעבירים ל- n יות כדי לעבוד עם מטריציוניות.

13.7.2 טענות

  1. C i ( [ T C B ] ) = T C B ( e i ) . כלומר [ T ] C B =( ∣ ∣ [ T( b 1 ) ] C … [ T( b n ) ] C ∣ ∣ ) .
  2. [ T ] C B ⋅ [ v ] B = [ T( v ) ] C .
  3. לכל v ¯ ∈ V , [ v ¯ ] B ∈ Sols ( [ T ] C B ) ⟺ v ¯ ∈ ker ( T ) .
  4. לכל u ¯ ∈ U , [ u ¯ ] C ∈ Cols ( [ T ] C B ) ⟺ u ¯ ∈ Im( T )
    לכן מסיקים ש- N ( [ T ] C B ) = dim ( ker ( T ) ) , Rank ( [ T ] C B ) = dim ( Im( T ) ) .
  5. T הפיכה ⟺ T C B הפיכה ⟺ [ T ] C B הפיכה, בנוסף ( [ T ] C B ) -1 = [ T -1 ] B C .
  6. [ S ○ T ] D B = [ S ] D C ⋅ [ T ] C B .
אלגוריתם לחישוב המטריצה המייצגת: נתונה העתקה T:V → U , בסיסים B=( b 1 , … , b n ) של V ו- C=( c 1 , … , c n ) של U . נרצה לחשב [ T ] C B באופן יעיל.
נבחר בסיס שנוח לחשב בו קואורדינטות ב- U - בדרך כלל הבסיס הסטנדרטי. W=( w 1 , … , w n ) . ( ∣ ∣ ∣ ∣ [ u 1 ] W … [ u m ] W [ T( b 1 ) ] W … [ T( b n ) ] W ∣ ∣ ∣ ∣ )→ R 1 → … … → ( 1 ∣ ⋱ ∣ [ T ] C B 1 ∣ ) באופן דומה לאיך שמחשבים מטריצה הופכית.
מטריצות שינוי הקואורדינטות: יהיו B,C שני בסיסים של מ”ו V . אז נגדיר את מטריצת שינוי הקואורדינטות מ- B ל- C על ידי: [ I d V ] C B .
  1. v ¯ ∈ V , [ I d V ] C B ⋅ [ v ¯ ] B = [ v ¯ ] C .
  2. [ T ] C B = [ Id ] C C ′ ⋅ [ T ] C ′ B ′ ⋅ [ I d V ] B ′ B .
הגדרה: יהיו A,B ∈ M n ( F ) , נאמר כי A ו- B דומות אם קיימת מטריצה הפיכה P כך ש- A= P -1 ⋅ B ⋅ P .
משפט: נתון A,B ∈ M n ( F ) ריבועיות, הבאים שקולים:
  1. A,B דומות.
  2. קיימת העתקה לינארית T:V → V ובסיסים C, C ′ של V כך ש- [ T ] C =A, [ T ] C ′ =B .
  3. כל העתקה לינארית T:V → V , אם קיים בסיס C של V כך ש- [ T ] C =A , אז קיים בסיס C ′ של V כך ש- [ T ] C ′ =B .
אם A,B דומות אז:
  1. Rank ( A ) = Rank ( B ) .
  2. N ( A ) = N ( B ) .
  3. tr ( A ) = tr ( B ) כאשר tr ( A ) = ∑ i=1 n ( A ) i,i .
  4. det ( A ) = det ( B ) .
  5. … נראה עוד בלינארית 2.

14 אלגוריתמים

14.1 צמצום סדרה לבת”ל

14.1.1 לפי שורות

יהיו v 1 , … , v n ∈ F m . נשים את v 1 , … , v n כשורות, B=( v 1 t ⋮ v n t ) ∈ M n × m ( F ) , ונדרג בלי להחליף שורות )כלומר, בעמודה של מקדם הפותח יש רק 1 במקום אחד והשאר אפסים, אבל המקדמים הפותחים לא ממוינים(. השורות שהתאפסו מתאימות לוקטורים שהיו תלוים לינארית באחרים, וסדרת השורות שלא התאפסו הן סדרה בת”ל.

14.1.2 לפי עמודות

נשים את v 1 , … , v n כעמודות, A=( v 1 … v n ) , נדרג ונבדוק שיש רק פתרון טריויאלי למערכת ההומוגנית ( A ∣ 0 ) , כלומר שאין אף משתנה חופשי.

14.2 השלמה של סדרה בת”ל לבסיס

תהא ( v 1 , … , v k ) סדרה בת”ל, ו- ( u 1 , … , u m ) סדרה פורשת. נבנה מטריצה שעמודותיה: ( v 1 , … , v k , u 1 , … , u m ) נדרג את המטריצה, ונסתכל על העמודות מ- u שנפתחה בהן מדרגה. את ה- u ים המתאימים נוסיף לסדרת ה- v ים, ונקבל בסיס.
